# kg_LayerColorTransition.jsx

**レイヤーカラートランジション**

レイヤーの**登場・退場に合わせて色を遷移**させるスクリプトです。
シェイプレイヤーの Fill 色と、その他のレイヤー（Fill エフェクト / 色被り補正）の両方に対応しています。

---

## インストール

`.jsx` を `Scripts` フォルダに配置し、`ファイル > スクリプト` から実行します。

---

## 動作

| タイミング | 動作 |
|-----------|------|
| 登場 | `inPoint` から「遷移時間」秒かけて **カラー A → カラー B** |
| 退場 | `outPoint` の「遷移時間」秒前から **カラー B → カラー A** |

---

## レイヤー種別ごとの処理

### シェイプレイヤー
シェイプ内のすべての **Fill** プロパティに HSL ベースのカラー遷移エクスプレッションを設定します。
A→B→A のループサイクルで自然な色変化を実現。

### その他のレイヤー
実行時ダイアログで方式を選択します。

| 方式 | 説明 |
|------|------|
| 塗り | Fill エフェクトで A→B→元の色 を遷移（コンポジットオプションで元の色を露出） |
| 色被り補正 | Tint エフェクト × 2 で遷移。Tint A の量でフェードイン、Tint B の量でフェードアウトを制御 |

---

## エフェクトコントロール

| パラメータ | 説明 |
|-----------|------|
| カラー A | 登場・退場時の色 |
| カラー B | 通常表示時の色 |
| 遷移時間 | 遷移にかける秒数 |
| 色を反転 | A と B を入れ替え |

---

## 使い方

```
1. レイヤーを選択（複数可）
2. スクリプトを実行
3. （シェイプ以外の場合）方式を選択 → OK
4. エフェクトコントロールで色・遷移時間を調整
```

---

## 注意事項

- **複数選択時**はヌルレイヤーを生成して一括制御します。
- **単体選択時**はレイヤー自身にエフェクトを追加します。
