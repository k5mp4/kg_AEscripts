# kg_guideShape.jsx

**ガイドシェイプ生成**

コンポジションのガイドラインで区切られた**各セルをシェイプレイヤーとして自動生成**するスクリプトです。
UI なし・即座実行。

---

## インストール

`.jsx` を `Scripts` フォルダに配置し、`ファイル > スクリプト` から実行します。

---

## 動作

1. アクティブなコンポジションの水平・垂直ガイドを読み取る。
2. コンプ端 + ガイド位置でグリッドを構成し、各セルのサイズ・位置を計算。
3. セルごとに長方形シェイプレイヤーを生成。

生成されるレイヤー名は `Guide Shape 1 [col,row]` の形式です。

---

## 使い方

```
1. コンポジションにガイドを設定（表示 > ガイド / ガイドを追加）
2. そのコンポジションをアクティブにしてスクリプトを実行
```

---

## 注意事項

- ガイドが存在しない場合はコンプ全体を 1 セルとして扱います（ガイドなし = セル 0 の状態）。
- ガイドが 1 本もない場合はコンプ = 1 セルなので、シェイプは 1 枚だけ生成されます。
- AE 16.1 (CC 2019) 以降で利用可能な `comp.guides` API を使用しています。
