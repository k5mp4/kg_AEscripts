# kg_advance_cornerpin.jsx

**拡張コーナーピン / シェイプパスヌルリンク** — v1.2.0

レイヤー種別を自動判定し、ヌルを生成してパラメータを制御するスクリプトです。
UI なし・即座実行。

---

## インストール

`.jsx` を `Scripts` フォルダに配置し、`ファイル > スクリプト` から実行します。

---

## 使い方

```text
1. 制御したいレイヤーを選択
2. スクリプトを実行
3. 生成されたヌルを移動してコーナー / 頂点を制御
```

---

## レイヤー種別ごとの動作

### シェイプレイヤー以外（映像・プリコンプ・テキスト等）

1. **Corner Pin** エフェクトを適用（既存があれば再利用）
2. 4 コーナーの位置にヌルを生成
3. 各コーナーピンプロパティがヌルの Position を参照するエクスプレッションを設定

| ヌル名 | コーナー | ラベル色 |
| --- | --- | --- |
| `CP_L{n}_UL` | 上左 | シアン |
| `CP_L{n}_UR` | 上右 | ブルー |
| `CP_L{n}_LL` | 下左 | ピーチ |
| `CP_L{n}_LR` | 下右 | ラベンダー |

### シェイプレイヤー

1. コンテンツ内の**全ベジェパス**を再帰的に収集
2. 各パスの頂点ごとにヌルを生成
3. パスプロパティに `fromComp()` を使ってヌル Position を追従するエクスプレッションを設定

| ヌル名 | 説明 |
| --- | --- |
| `SP_L{n}_P{p}_V{v}` | レイヤー n のパス p の頂点 v |

---

## 注意事項

- ヌルレイヤー自身を選択しても処理されません
- シェイプレイヤーで対応するのは**ベジェパス**のみ（矩形・楕円などのパラメトリックシェイプは対象外）
- シェイプグループに独自のトランスフォームが掛かっている場合、ヌルの初期位置がずれることがあります
- 再実行すると既存の同名ヌルを削除して再生成します
- ヌルは非シャイ状態で生成されます
- 3D レイヤーには対応していません（2D のみ）
