# kg_timeremap.jsx

**タイムリマップグリッド**

動画レイヤーをグリッド状に分割または縮小タイルとして並べ、各タイルの再生タイミングを**パターンに応じてずらす**スクリプトです。タイムリマップエクスプレッションを自動生成します。

---

## インストール

`.jsx` を `Scripts` フォルダに配置し、`ファイル > スクリプト` から実行します。

---

## 設定項目

| 項目 | 説明 |
|------|------|
| 列数 (Columns) | グリッドの列数 |
| 行数 (Rows) | グリッドの行数 |
| 最大オフセット秒数 (Max Offset) | 最も遅いタイルの再生遅延（秒） |

### グリッドモード

| モード | 説明 |
|--------|------|
| Split（分割） | 動画を各タイルサイズに切り抜いて並べる |
| Tile（縮小タイル） | 動画全体を縮小して敷き詰める |

### オフセットパターン

| パターン | 説明 |
|---------|------|
| Diagonal | 左上→右下の対角線順にディレイが増加 |
| Radial | グリッド中心から外側にかけてディレイが増加 |
| Left→Right | 列方向（左→右）にのみディレイが増加 |
| Gradient Image | 指定レイヤーの輝度値でディレイを制御 |

**Gradient Image パターン**使用時は「グラデーションレイヤー名」にコンプ内のレイヤー名を入力してください。タイル中心をそのレイヤー上でサンプリングしてディレイを決定します。

---

## 使い方

```
1. タイムラインで動画レイヤーを選択
2. スクリプトを実行
3. ダイアログで列数・行数・最大オフセット・モード・パターンを設定 → OK
```

---

## 注意事項

- タイムリマップが有効になっていないレイヤーには自動でタイムリマップを有効化します。
- Gradient Image パターンでは、指定したレイヤーがコンプに存在している必要があります。
