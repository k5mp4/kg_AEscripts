# kg_MaskUtil.jsx

**マスクユーティリティ** — v1.4.0

マスクパスの整形・Wiggle アニメーション・リンクヌル作成をまとめたパネル型スクリプトです。

---

## インストール

`.jsx` を `ScriptUI Panels` フォルダに配置し、AE 再起動後に `ウィンドウ` メニューから開きます。

---

## 機能一覧

### Select Mask Paths
選択レイヤーの**全マスクパスプロパティ**を選択状態にします。
タイムラインで Mask Shape をまとめてコピー／編集するときに便利です。

### Fit Mask to Comp
マスク頂点を**コンポジション境界内にクランプ**します。
現在時刻のレイヤートランスフォームを基準に `layerToComp → clamp → compToLayer` 変換を行うため、回転・スケールが掛かったレイヤーでも正確に動作します。

---

### Setup Wiggle (with Null)
各マスク頂点に **MW_L\*_M\*_V\*** というヌルを生成し、ヌルの Position エクスプレッションで `noise()` を使ってカクカク動作させます。

| 設定項目 | 説明 |
|---------|------|
| posterizeTime | フレーム更新レート (fps) |
| strength | 振れ幅 (px) |
| seed | ランダムシードのベース値 |

- **MW_Control** ヌルのエフェクトコントロールでパラメータを一括管理。
- 各頂点は**放射方向**にのみ動くため、マスクが溶けるような動きを防ぎます。
- 3D レイヤー対応（ヌルも 3D 化して Z 座標を保持）。

### Setup Wiggle (without Null)
ヌルを作成せず、**マスクパスエクスプレッションだけ**で Wiggle を実現します。
レイヤー自身のエフェクトコントロールにスライダーを追加します。
ヌル方式よりシーンがシンプルに保てますが、個別頂点の可視化はできません。

### Create Linked Nulls
Wiggle なしで各頂点に **ML_L\*_M\*_V\*** ヌルを作成し、マスクパスをそのヌル Position に追従させます。
- ヌルにキーフレームを打ってマスクを動かしたり、他レイヤーへピックウィップしたりと汎用的に使えます。
- ヌルはデフォルトで非シャイ（操作しやすいよう表示）。

### Wiggle 停止 / 再開
選択レイヤーの Wiggle エクスプレッションをまとめて**有効 / 無効**切り替えます。

| 方式 | 停止後の手順 |
|------|------------|
| without-null | マスクパスを直接編集 → 再開で編集後の頂点が新しい基点になる |
| with-null | ヌル / マスクを編集 → 「Setup Wiggle (with Null)」を再実行して焼き直す |

### Remove Wiggle / Linked Nulls
Wiggle ヌル・リンクヌル・マスクパスエクスプレッションをすべて削除します。
他の MW_/ML_ ヌルがなくなった場合は `MW_Control` レイヤーも自動削除されます。

---

## 使い方

```
1. レイヤーを選択
2. パネルの各ボタンを実行

Wiggle セットアップの場合:
  posterizeTime / strength / seed を入力 → ボタン実行
```

---

## ヌル命名規則

| プレフィックス | 説明 |
|--------------|------|
| `MW_Control` | Wiggle パラメータ管理ヌル |
| `MW_L{n}_M{m}_V{k}` | Wiggle with Null の頂点ヌル |
| `ML_L{n}_M{m}_V{k}` | Linked Nulls の頂点ヌル |
