# kg_ShapeEffectAnimator.jsx

**シェイプエフェクトアニメーター** — v1.0.0

シェイプレイヤーのエフェクト（Path Offset / Trim Paths / Twist / Pucker & Bloat）に**登場・退場アニメーション**を自動設定するパネル型スクリプトです。

---

## インストール

`.jsx` を `ScriptUI Panels` フォルダに配置し、AE 再起動後に `ウィンドウ` メニューから開きます。

---

## 機能一覧

### 適用エフェクト

| エフェクト | 説明 |
|-----------|------|
| パスのオフセット | 量: で膨縮量を指定 |
| パスのトリミング | 開始点 / 終了点を個別に選択可 |
| 旋回 | 旋回量を指定 |
| パンク膨張 | パンク膨張量を指定 |

複数エフェクトを同時に有効にすると、すべてに一括適用されます。

### アニメーション設定

| 項目 | 説明 |
|------|------|
| フレーム数 | アニメーションの長さ（フレーム） |
| ずらしアニメーション | チェックするとパスごとに 1F ずつタイミングをずらして表示 |

### アニメーション動作

- **登場**: `inPoint` からフレーム数かけて「開始値 → 0」
- **退場**: `outPoint` からフレーム数前より「0 → 開始値」
- キーフレームはエフェクトコントロールのスライダー経由で管理されるため後から調整可能

---

## 使い方

```
1. シェイプレイヤーを選択
2. 適用したいエフェクトにチェック / 数値を入力
3. フレーム数・ずらしを設定
4. 「適用」ボタンをクリック
```

---

## 注意事項

- シェイプレイヤー以外には適用できません。
- パスのトリミングを使う場合は「開始点」または「終了点」のいずれかを選択してください。
