# kg_bufferPrecomp.jsx

**バッファ付きプリコンポーズ** — v1.1.0

選択レイヤーを**前後 120 フレームのバッファ付き**でプリコンポーズするスクリプトです。
モーションブラーや Time Remapping の影響が端で切れるのを防ぐために使います。

---

## インストール

`.jsx` を `Scripts` または `ScriptUI Panels` フォルダに配置し、`ファイル > スクリプト` から実行します。

---

## 動作

1. 選択レイヤーの時間範囲（inPoint〜outPoint）に前後 **120 フレーム**のバッファを加算。
2. その期間を持つ新しいコンポジションとしてプリコンポーズ。
3. 親コンプではプリコンプレイヤーが元の位置に配置されます。

---

## 使い方

```
1. プリコンポーズしたいレイヤーを選択
2. スクリプトを実行
3. ダイアログでプリコンプ名を入力 → OK
```

---

## 注意事項

- バッファ量は固定で 120 フレームです（変更する場合はスクリプト先頭の `BUFFER_FRAMES` を編集してください）。
- 複数レイヤーを選択すると、それらをまとめて 1 つのプリコンプにまとめます。
